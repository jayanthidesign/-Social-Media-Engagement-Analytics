# Social Media Engagement Analytics

# Project Overview:

This project analyzes social media engagement data using Python to uncover patterns in user
behavior, content performance, and sentiment. Visualizations and statistical summaries are used to
support actionable insights.

# Tools & Technologies:

Language: Python 

# Data Cleaning & Preparation:

Handled missing values in age,likes,comment,share,gender and sentiment column.
    
Fixed incorrect data types.
    
Standardized gender label.

# Exploratory Data Analysis (EDA):

Reels generate the highest total engagement (≈ 16.06M)

The tech category has the highest engagement (≈ 8.44M)

Brazil records the highest average engagement rate (1.54)

# Key Insights:

Users aged 30–39 generate the highest average engagement score, followed by the <20 age
group. Engagement gradually decreases in older age groups, with 50+ users showing the lowest
engagement.

Non-verified accounts show slightly higher engagement scores than verified accounts, suggesting
that audience interaction is not necessarily dependent on account verification.

The dataset shows the highest average impressions around hour 0 (midnight) with approximately
50,013 impressions, suggesting that late-night posting may drive strong visibility.

Mobile devices have the highest average watch time (≈4087 sec), compared to desktop and
tablet, indicating that most users consume content primarily through mobile devices

Although neutral posts are more common, negative posts generate slightly higher engagement,
indicating that emotionally strong content tends to attract more audience interaction even if it appears
less frequently.

# Recommendations:

Content type, category, sentiment, and posting time all play an important role in determining engagement
levels. Reels and tech-related content attract the most attention, while mobile users contribute the
highest watch time. These insights can help organizations and content creators make data-driven
decisions to improve content performance and audience engagement.
